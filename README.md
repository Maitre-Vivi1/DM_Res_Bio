## Quickstart

- Create a virtual environment : `python3 -m venv venvBioSys`
- Activate it : `source venvBioSys/bin/activate`
- Install dependencies : `pip install -r requirements.txt`
- To run a unittest : `python -m unittest path/testFile.py`
